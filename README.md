## Multimode quantum light in waveguides with detectors. Entanglement, squeezing and other properties.

Python 3.5+

Set it up:
```sh
python3 -m venv ENV
source ENV/bin/activate
pip3 install -r requirements.txt
```

Run tests:
```sh
pytest
```

Usage:
```sh
python3 main.py --det FIRST --phase 0.0
```
